# TACA test programs

`github_workflow.tq`
- Main end-to-end sample with imports, functions, loops, conditionals, strings, chars, bools, arithmetic, bitwise ops, compound assignments, postfix ops, and builtins like `println` and `type`.

`pointer_coverage.tq`
- Focused sample for address-of, dereference, pointer assignment, and pointer compound assignment.

`operator_coverage.tq`
- Focused sample for arithmetic, power, shifts, bitwise, comparison, and logical operators.

`builtin_coverage.tq`
- Focused sample for builtin functions `alloc`, `calloc`, `realloc`, `memcpy`, `memncpy`, `rm`, and `type`.

`github_helpers.tq` + `github_labels.tq`
- Imported support modules used by `github_workflow.tq`.
