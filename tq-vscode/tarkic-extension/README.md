# Tarkic Extension

Syntax highlighting (TextMate grammar) + language configuration for the Tarkic language (`.tq`).

## Features

- Keywords (`if`, `else`, `for`, `fn`, `ret`, `return`, etc.)
- Types (`i32`, `u64`, `f64`, `bool`, `str`, `void`, etc.)
- Numbers, strings, operators
- Line comments `'' ...` and block comments `''' ... '''`
- Bracket/quote auto-closing + surrounding pairs

## Requirements

None.

## Usage

- Install the extension (VSIX or Marketplace).
- Open any `.tq` file (language id: `tarkic`).

## Known Issues

- This extension provides syntax highlighting only (no IntelliSense/LSP yet).
